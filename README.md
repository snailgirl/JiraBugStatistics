# JiraBugStatics 
Jira bug 解决率统计

## 配置信息
- conf.ini：
    - email 配置接收bug统计报告的邮箱，多个邮箱以`,`隔开
    - module 配置需要统计bug的项目模块，多个模块以`,`隔开
- jira.yaml：
    - 项目模块配置
    - gitlab 项目名称
    - gitlab 项目key
    - 单个模块统计项目显示
    - 总体统计项目显示
- mail.ini：
    - 配置发件箱相关信息
## 使用
```
python3 statistics.py
```
